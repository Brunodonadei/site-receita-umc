# site-receita-umc

Um site de receitas simples e moderno feito para organizar e exibir diversas receitas de forma prática.

O projeto permite:

- Listar receitas com imagem, título e descrição.
- Acessar detalhes de cada receita em páginas individuais.
- Armazenar receitas em um arquivo JSON para fácil manutenção e expansão.
- Suporte a filtros ou navegação por `slug` nas URLs.

Feito com HTML, CSS e JavaScript puro.
